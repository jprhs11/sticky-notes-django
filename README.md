# 📝 Sticky Notes Django Application

A simple, responsive sticky notes application built with Django. This project allows users to create, read, update, and delete (CRUD) digital sticky notes, mimicking a physical notice board.

## 🚀 Features
- **Full CRUD Functionality**: Create, View, Edit, and Delete notes.
- **Responsive Design**: Styled with Bootstrap to work on desktop and mobile.
- **Automated Testing**: Includes unit tests for models and views to ensure reliability.
- **MVT Architecture**: Built using Django's Model-View-Template pattern.

## 🛠️ Tech Stack
- **Backend**: Django (Python)
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Database**: SQLite (Development) / Compatible with MariaDB
- **Design**: UML (Use Case, Sequence, and Class Diagrams)

## 📂 Project Structure
- `sticky_notes/`: Project configuration and settings.
- `notes/`: Main application logic (Models, Views, Forms).
- `static/`: Custom CSS for the "sticky" visual effect.
- `templates/`: HTML templates for the UI.
